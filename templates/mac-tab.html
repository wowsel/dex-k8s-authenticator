Prerequisites: kubectl + kubelogin

You need both kubectl and kubelogin installed. Follow the instructions below.

0

Install Required Tools

Install kubectl

The easiest way on macOS is via Homebrew:

brew install kubectl

Or see official documentation.

Install kubelogin

Choose one of these methods:

Homebrew (Recommended)
brew install kubelogin
Krew
kubectl krew install oidc-login
Go Install
go install github.com/int128/kubelogin@latest

Or download from GitHub Releases:

# For Apple Silicon (M1/M2/M3)
curl -LO https://github.com/int128/kubelogin/releases/latest/download/kubelogin_darwin_arm64.zip && \
unzip kubelogin_darwin_arm64.zip && \
chmod +x kubelogin && \
sudo mv kubelogin /usr/local/bin/kubectl-oidc_login
# For Intel Mac
curl -LO https://github.com/int128/kubelogin/releases/latest/download/kubelogin_darwin_amd64.zip && \
unzip kubelogin_darwin_amd64.zip && \
chmod +x kubelogin && \
sudo mv kubelogin /usr/local/bin/kubectl-oidc_login
